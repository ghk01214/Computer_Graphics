화면의 하단에 3행 8열의 네모 칸이 만들어진다.

생성되는 도형은 삼각형과 사각형이다.

도형은 5초에 1개씩 새로 생성된다.

도형은 생성 위치에 따라 좌상, 좌하, 우상, 우하 4개의 방향 중 한 방향으로 움직인다.

날아오는 도형을 가르면 정방향의 삼각형 2개로 쪼개진다.(자유형 X)

L키를 누르면 도형이 GL_LINE 모드로 출력된다.

F키를 누르면 도형이 GL_FILL 모드로 출력된다.

도형의 경로는 기본적으로 표시되지 않으며 p키를 누르면 표시되고 다시 p를 누르면 경로가 사라진다.

+키를 누르면 도형의 이동속도를 증가시키고 -를 누르면 이동속도를 감소시킨다.

q를 누르면 프로그램이 종료된다.

마우스로 클릭 드래그를 하면 자르는 단선이 마우스를 따라서 그려진다.

떨어진 도형이 화면 아래의 칸에 들어가는 부분은 미구현.

마우스로 저장된 조각 이동 부분 미구현.
